Creating Map 19

Build Visual = Choose the visual map



  1.  Select "State/Country" from AirCrashFatalities and drag it to Location.
  2. Select " CrashType" from AirCrashFatalities and drag it to Legend. Legend title = Off.
  3. Select "(DeceasedPerYear)" from AirCrashFatalities and drag it o Bubbles size. Add    tooltips the following "(GroundYoY%)", " (DeceasedYoY%)". "(CollideYoY%)"
  4. Format Visual, Map settings, Style = Road, Show labels = On, Controls, Auto zoom = Off, Zoom Buttons = On, Lass button = Off, Geocoding culture = Auto.
    Bubbles size 15px. Colors, Collide =#73361C, Ground =#68FEFC.
  5. General. Height = 284, Width =529, Position, Horizontal = 751, Vertical = 81, Title = On, Text = " Location", Font type = Segoe Ui, Semibold,  size =15, Text color = #000000, Background = #D8D7BF, Align to center, Text wrap = Off,Tooltips, On Type = Report page, Page = Auto, Background color = #FFFFF0, Transparency 23%.
