Creating DAX measures and Refresh Date 14





Create a DAX Measure follow these steps:

- Select the table "Measure"

- Right Click and select New Measure

- Write the following DAX Measures on the space provide one at a time and click the right mark


1.   (TotalGround) = IF(ISBLANK('Measure'[Ground]),0,'Measure'[Ground])

2. Ground = CALCULATE (COUNTROWS ( AirCrashFatalities ),KEEPFILTERS (TREATAS

   ( { "Ground" }, AirCrashFatalities[CrashType] )))

3.  (TotalCollide) = IF(ISBLANK('Measure'[Collide]),0,'Measure'[Collide])

4. Collide = CALCULATE(COUNTROWS ( AirCrashFatalities ),KEEPFILTERS

   (TREATAS ( { "Collide" }, AirCrashFatalites[CrashType] )))

5. (Ground) = IF(ISBLANK('Measure'[Ground]),0,'Measure'[Ground])

6. (Colide) = IF(ISBLANK('Measure'[Collide]),0,'Measure'[Collide])

7.  ( DeceasedPerYear) = IF(ISBLANK('Measure'[Deceased_per_Yr]),0,'Measure'[Deceased_ per_Yr])

8. Deceased_per_ Yr = DIVIDE([Total_Deceased],DISTINCTCOUNT('Date'[Year])+0)

9. Total_Deceased = SUM(AirCrashFatalities[Fatalities])

10.   Arrow Ground = SWITCH(TRUE(), [Ground_YoY%] <0, UNICHAR(9660), UNICHAR(9650))

11. (GroundYoY%) = IF(ISBLANK('Measure'[Ground YoY%]),0,'Measure'[Ground YoY%])

12.Ground YoY% = DIVIDE([Ground]-[Ground LY],[Ground LY],0) 

13.Ground LY = CALCULATE([Ground], DATEADD('Date'[Date],-1,YEAR))

14. Arrow Collide = SWITCH(TRUE(), [Colide_YoY%] <0, UNICHAR(9660), UNICHAR(9650))

15.(ColideYoY%) = IF(ISBLANK('Measure'[Colide YoY%]),0,'Measure'[Colide YoY%])

16.Colide YoY% = DIVIDE(Collide]-[Colide LY],[Colide LY],0)

17.Colide LY = CALCULATE([Collide], DATEADD('Date'[Date],-1,YEAR))

18.Arrow Deceased YoY% = SWITCH(TRUE(), [Deceased YoY%] <0, UNICHAR(9660), UNICHAR(9650))

19.(DeceasedYoY%) = IF(ISBLANK('Measure'[Total_Deceased]),0,'Measure'[Total_Deceased])

20.Deceased_YoY% = DIVIDE([Total_Deceased]-[TotalDeath_LY],[TotalDeath_LY],0)

21.TotalDeath_LY = CALCULATE([Total_Deceased], DATEADD('Date'[Date],-1,YEAR))

Refreshed Date



Steps

1. Click on Transform Data found under Home tab.

2. Click on New Source - Blank query

3. On the space provided write =  DateTime.LocalNow() and hit the check mark

4. Then we gave a name (Data Refresh) for our table under the properties

5. Click "To Table" found under file & rename the column to " Date Time"

6. Duplicate the actual column and rename it "Date". After that go to "Transform" and under date click "Date Only"

7. Duplicate the actual column and rename it "Time". Then go to transform and under Time click "Time Only"

8. Close and Apply.
