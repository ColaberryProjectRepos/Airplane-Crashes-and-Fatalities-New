Adding Collaberry Logo

1. Click on insert from ribbon.

2. Click on Image in Elements

3. Select the Image from the Extracted file

4. Set General: Properties, Heights = 60 and Width = 213 Position, Horizontal = 0, Vertical = 0

Adding Project Title


  1. Click on the Text box from Home or insert ribbon. Type the name of the project "AIRPLANE CRASH FATALITIES (1908-2009)" 
  2. Select the text then change the font size to 18 and font type Lucida Sans Unicode. Select the text then Alignment to center.
  3. General: Properties, Height = 61, and Width = 534, Position, Horizontal = 225, Vertical = 0. Select Effect then turn off the background to =0

      Follow the Gif bellow for step by step



Refreshed Date Setup.


  1. Open Visualization Pane, Select Visual Card , Drag the visual to the dashboard. Click on refreshed date from Data Refresh Table and place it inside the fields. 
  2. General: Properties, Height = 68, Width = 217, Position Horizontal = 740, Vertical = 7
  3. Visual: Callout Value, Color = #000000, Font size 12, Font type Segoe UI Semibold, Category = Off
  4. General Title On, Text Title = " Refreshed Date", Font Color = #252423, Background Color no fill, Font size = 12,  Horizontal alignment to middle.
