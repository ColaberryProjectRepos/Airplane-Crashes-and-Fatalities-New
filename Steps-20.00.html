Adding Word Cloud to visual



  1. Select on 4 dots under Build Visual, then add more visuals
  2. Under search box type "Word" then add the Word cloud to Visual.
  3. Build Visual: Category, "Type", Values "CrashType", Tooltip, "CollideYoY%","DeceasedYoY%", "GroundYoY%
  4. General. Height = 303, Width =521, Position, Horizontal = 237, Vertical = 363, Title = On, Text = " Crash Type", Font type =Segoe  Ui Semibold,  size =15, Text color  =   #000000, Background = #D8D7BF, Align to center, Text wrap = Off. Tooltips, On Type = Report page, Page = Auto, Background color = #FFFFF0,Transparency 0%.
