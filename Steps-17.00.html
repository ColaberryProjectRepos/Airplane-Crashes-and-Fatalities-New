Adding Scroller Visual 17

Scroller Visual


  1. Select on 4 dots under Build Visual, then add more visuals
  2. Under search box type "Scroller" then add the scroller to Visual.
  3. Build Visual, Category, "Country/State", Measure absolute "Country/State", Measure deviation, "(DeceasedPerYr)"
  4. Visual, Status indicator = On, Status indicator coloring = On, Status text coloring Off,  Font size =20, Scroll speed = 1.2, Text color #744EC2, Background #FFFFF0, Update interval = 50, Effect, Background = On, Color  =  # FFFFF0, Transparency = 0%, Shadow = Off, Header icon = Off.
  5. General: Properties Height = 52, Width = 1280, Position, Horizontal = 0, Vertical = 668
