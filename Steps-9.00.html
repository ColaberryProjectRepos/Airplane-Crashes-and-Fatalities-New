Importing Data to your computer and downloading the file to power bi desktop 9

Click the following link

https://www.kaggle.com/datasets/thedevastator/airplane-crashes-and-fatalities

 Download the file and save it to folder (Rename the file name to AirCrashFatalities)



Downloading the AirCrashFatalities file to Power bi Desktop.

  1.  Open Power bi Desktop
  2.  Click Get data on power bi desktop
  3.  Under Get data select Text/CSV file and open the file from the folder you saved it to your   computer 
  4.  Click on the load button to load the data
